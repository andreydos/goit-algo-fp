## Task 7

File: `task7_monte_carlo_dice.py`

This program simulates rolling two dice many times.

Run:

```bash
python3 task7_monte_carlo_dice.py
```

## Monte Carlo Conclusion

The Monte Carlo results are close to the analytical probabilities.
The more rolls we use, the closer the simulated values are to the expected values.
The most common sum is 7, and the least common sums are 2 and 12.
